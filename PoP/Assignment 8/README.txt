HOW TO RUN THE CODE:

This /src-folder contains the following file:
 * 8i0.fsx

They are run in the following manner:

First, the ImgUtil library must be compiled into a .dll file and placed in the /src-folder.

Afterwards the rest of the files can be compiled and run in the following manner
"fsharpc -r img_util.dll 8i0.fsx"
"mono 8i0.exe"
